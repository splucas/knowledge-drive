# Knowledge Drive
Simple edutainment POC built in Unity where players drive down the streets of 
knowledge and match topics to related topic texts.  After 5 correct answers or
3 incorrect answers, it's game over!

### Requirements
- Unity 2022 
- Packages:
    - TextMesh Pro 
    - Input System
